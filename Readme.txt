 - convert WGS84 coordinates (decimal degrees) to DMS format (Degrees, Minutes, Seconds)


+ WGS84 Coordination:
WGS84 (World Geodetic System 1984) is the standard coordinate system used by GPS.
It expresses geographic coordinates as decimal degrees, like this:

    Latitude: 40.748817

    Longitude: -73.985656

So a location is represented as:
(40.748817, -73.985656) — e.g., a point in New York City.

+ DMS format:
DMS (Degrees, Minutes, Seconds) is a more traditional way to express geographic coordinates.

The same point becomes:
Latitude: 40° 44' 55.74" N
Longitude: 73° 59' 8.36" W
